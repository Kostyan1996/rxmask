Phone mask - specify mask, mask symbol and allowed symbols

class="rxmask" mask="***-**-**" symbol="*" allowedSymbols="[0-9]"

Phone mask - showing whole mask including unfilled part

class="rxmask" mask="+7 (___) ___-__-__" symbol="_" showMask="true" allowedSymbols="[0-9]"

Phone mask - only part of mask is shown

class="rxmask" mask="+7 (___) ___-__-__" symbol="_" showMask="8" allowedSymbols="[0-9]"

Phone mask, but only for special characters - allowedSymbols can include symbols from mask EXCEPT "symbol" property itself (note that special symbols should be escaped)

class="rxmask" mask="+7 (___) ___-__-__" symbol="_" showMask="true" allowedSymbols="[ \-\+\(\)]"

Regex mask - only certain symbols are allowed for each character (you don't need to specify "mask" property if "rxmask" is present (it will be ignored))

class="rxmask" rxmask="[A-Z][A-Z] [0-4][\d][\d]-[a-z][\d]" symbol="*" showMask="true"
